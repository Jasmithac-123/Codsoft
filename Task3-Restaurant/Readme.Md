# 🍽️ Restaurant Menu UI/UX Design

## 📖 Project Overview
This project is a simple Restaurant Menu UI/UX design created using **Figma**. The design focuses on providing a clean, attractive, and user-friendly interface that allows customers to easily browse menu items and place orders.

## 🎯 Objective
- Design a visually appealing restaurant menu.
- Create an easy-to-use interface for customers.
- Organize menu items into categories for better navigation.

## ✨ Features
- Clean and modern user interface
- Category-based menu (Pizza, Burger, Pasta, Drinks, Desserts)
- Search bar for finding dishes
- Food cards with image, description, and price
- Add to Cart button
- Simple navigation bar

## 🛠️ Tools Used
- Figma
- Google Fonts (Poppins)
- Icons (Iconify/Figma Community)

## 🎨 Design Principles
- Visually appealing layout
- Easy navigation
- Categorized menu items
- Consistent colors and typography
- Beginner-friendly design
