# 📱 Mobile App Signup Flow

A modern and user-friendly Mobile App Signup Flow designed in Figma. This project provides a smooth onboarding experience with a clean interface and easy navigation.

## ✨ Features
- Welcome Screen
- Create Account
- Personalization
- Email Verification (OTP)
- Account Success Screen

## 🎯 UI/UX Principles
- Short and Sweet
- Personalized
- Useful
- Error-Free
- Secure

## 🛠️ Tools Used
- Figma
- Material Icons
- Storyset Illustrations
